    Here are 8 popular js libraries:

    1. Leaflet

      Leaflet is the leading open-source JS library for mobile friendly interactive maps.

    2. Polished

      Polished is a great lightweight toolset for writing styles in JS with Saas-style helper functions and mixins.

    3. Swiper

      Swiper is the modern free mobile touch slider with hardware accelerated transitions & amazing native behaviour.


    4. Data driven documents or d3.js

      Data driven documents or d3.js is a JavaScript library for manipulating documents based on data.


    5. Konva.js

      It's an HTML5 Canvas JavaScript library that extends the 2d context by enabling Canvas interactivity.

    6. Fullpage

      It helps you create full screen scrolling websites. It's easy to use and has many options to customize.

    In conclusion, JavaScript libraries are a valuable tool for web developers, providing a range of features and functionality that can greatly enhance the user experience and functionality of web applications.
